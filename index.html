Regras e publicação

Informações funcionais e técnicas mantidas dentro do próprio HTML.

Funcionalidades

  • Placares salvos automaticamente no navegador.
  • Classificação recalculada em tempo real.
  • Busca global e filtro por grupo trabalhando juntos.
  • Abas internas com link direto por hash.

Critérios

  • Vitória soma 3 pontos.
  • Empate soma 1 ponto para cada seleção.
  • Desempate por pontos, saldo de gols e gols pró.
  • As duas primeiras posições aparecem destacadas.

GitHub Pages

  • O projeto funciona sem backend.
  • O arquivo principal é este `index.html`.
  • Os dados ficam no `localStorage` do visitante.
  • A navegação usa hashes, como `#simulator`.
